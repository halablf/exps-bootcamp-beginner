Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using a system of tags and elements.

Key Concepts:

  • Tags: HTML elements are defined by tags, usually in pairs like <tag>content</tag>
  • Headings: Use <h1> to <h6> for different heading levels
  • Paragraphs: Use <p> tags to create text paragraphs
  • Document Structure: Every HTML document needs <html>, <head>, and <body> tags
