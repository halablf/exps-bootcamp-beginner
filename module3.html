Module 3: CSS Basics

Learn how to style your HTML pages with colors, fonts, and spacing using CSS.

Introduction to CSS

CSS (Cascading Style Sheets) is the language used to style and layout web pages. It controls the visual presentation of HTML elements.

Key Concepts:

  • Selectors: Target HTML elements to style them
  • Properties: Define what aspect to change (color, font, margin, etc.)
  • Values: Specify how to change the property
  • CSS Rules: selector { property: value; }

Common CSS Properties:

  • color: Sets text color (e.g., red, #FF0000, rgb(255,0,0))
  • font-family: Sets the font type (e.g., Arial, sans-serif)
  • font-size: Sets text size (e.g., 16px, 1.2em)
  • margin: Sets outer spacing around elements
  • padding: Sets inner spacing within elements
  • background-color: Sets background color

CSS Syntax Example:

h1 {
  color: blue;
  font-size: 24px;
  margin: 20px;
}
