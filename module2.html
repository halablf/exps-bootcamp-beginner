Adding Images and Links

Now that you know the basics, let's learn how to add images and create links to make your web pages more engaging and connected.

Key Concepts:

  • Images: Use <img> tag with src and alt attributes
  • Links: Use <a> tag with href attribute to create hyperlinks
  • Lists: Use <ul> (unordered) and <ol> (ordered) for lists
  • Attributes: Additional information that modifies HTML elements

Image Syntax:

<img src="image.jpg" alt="Description">

Link Syntax:

<a href="https://example.com">Click here</a>

List Syntax:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
